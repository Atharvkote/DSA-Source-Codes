## Note 📝 :

There is a Inbuilt Method for Array Sorting
```.sort(array);``` in Arrays Class

 ### Basic Syntax

 ```java
int[] array = new int[size];
Arrays.sort(array);
//Sorts the array,changes are done in the original array !
```
### Specs

- **Algorithms :** Quick Sort.
- **Time Complexity :** O(logn)
